## H2: そもそも確定申告って必要？判定フロー

まず、自分に確定申告が必要か確認しよう。

### 確定申告が必要な人

以下のどれか1つでも当てはまる人は**確定申告が必要**だよ。

| 状況 | 判定基準 |
|---|---|
| **専業（チャトレ・夜職が本業）** | 年間所得48万円超 |
| **本業あり+副業** | 年間副業所得20万円超 |
| **主婦（扶養内）** | 年間所得48万円超で扶養外 |
| **学生** | 年間所得48万円超 |

※「所得」=「収入」-「経費」のことだよ💡

### 具体例

**ケース①: 本業OL＋副業チャトレ**
- 副業収入: 年間60万円
- 経費: 年間15万円
- 副業所得: 45万円
- → **20万円超なので確定申告必要** ✅

**ケース②: 専業チャトレ**
- 年間収入: 200万円
- 経費: 40万円
- 所得: 160万円
- → **48万円超なので確定申告必要** ✅

**ケース③: 学生バイト＋在宅チャトレ**
- バイト給与: 年間80万円
- チャトレ収入: 年間15万円
- チャトレ経費: 3万円
- チャトレ所得: 12万円
- → **副業所得20万円以下なので所得税の確定申告は不要**
- ※ただし**住民税の申告は必要**（後述）

### ナビちゃんからの注意💡

「20万円以下なら申告不要」は**所得税だけの話**。住民税は金額に関係なく申告が必要な自治体が多いよ。

「申告不要＝ゼロ申告でOK」じゃなくて「住民税だけ申告」が正解のケースが多いの。注意してね📝

## H2: 所得税の仕組みと計算方法

確定申告って「去年1年間（1月〜12月）の所得にかかる税金を計算して、国に報告する手続き」のことだよ。

### 所得税の計算式

```
① 収入 - 経費 = 所得
② 所得 - 各種控除 = 課税所得
③ 課税所得 × 税率 - 控除額 = 所得税
```

### 所得税の税率表（2026年時点）

| 課税所得 | 税率 | 控除額 |
|---|---|---|
| 195万円以下 | 5% | 0円 |
| 195万〜330万円 | 10% | 97,500円 |
| 330万〜695万円 | 20% | 427,500円 |
| 695万〜900万円 | 23% | 636,000円 |
| 900万〜1,800万円 | 33% | 1,536,000円 |

### 控除の種類（よく使うもの）

| 控除名 | 金額 | 対象 |
|---|---|---|
| **基礎控除** | 48万円 | 全員 |
| **社会保険料控除** | 支払額全額 | 国保・年金を払っている人 |
| **医療費控除** | 10万円超の医療費 | 病気・歯科治療した人 |
| **生命保険料控除** | 最大12万円 | 生命保険加入者 |
| **扶養控除** | 38万円〜 | 子ども・親を扶養している人 |

### 計算例: 専業チャトレ（年収200万円）

```
収入: 200万円
経費: 40万円（通信費・機材・衣装など）
所得: 160万円

控除:
- 基礎控除: 48万円
- 国保料: 15万円
- 国民年金: 20万円
合計控除: 83万円

課税所得: 160万円 - 83万円 = 77万円

所得税: 77万円 × 5% = 38,500円
```

## H2: 経費にできるもの・できないもの一覧

チャトレ・夜職の経費は、**「お仕事に必要な支出」**ならOK。

### 経費にできるもの✅

| 項目 | 具体例 |
|---|---|
| **通信費** | Wi-Fi、スマホ代（業務使用分のみ） |
| **機材費** | PC、スマホ、三脚、照明、マイク、Webカメラ |
| **衣装費** | 撮影・出勤用の服、下着、コスプレ |
| **美容費** | 撮影前のヘアメイク、ネイル、エステ（業務前提） |
| **交通費** | お店への通勤、面接移動 |
| **家賃（按分）** | 自宅で配信する部屋の面積比 |
| **光熱費（按分）** | 配信時間分の電気代 |
| **書籍・研修** | 業界本、セミナー参加費 |
| **手数料** | サイト利用料、振込手数料 |

### 経費にできないもの❌

- プライベートの食事、旅行
- 家族へのプレゼント
- 趣味の買い物
- 業務と関係ない美容整形

### 家賃・光熱費の按分方法

**例**: 50m²のアパート、配信部屋8m²

- 家賃8万円 × (8÷50) = 12,800円/月 → 年間15.4万円が経費
- 電気代: 配信8時間 ÷ 24時間 = 33% → 月の電気代 × 33%

「どこまで按分できる？」→ 「業務で使う割合を合理的に説明できる範囲」と覚えてね💡

### ナビちゃんの節税アドバイス💓

領収書は**全部取っておく**のが鉄則。

- スーパーのレシート（衣装）
- Amazon注文履歴（機材）
- Uber Eatsの履歴（配信中の食事も部分的にOKなケースあり）

スマホで写真撮っておくだけでも、後で集計するとき助かるよ✨

## H2: 確定申告書の書き方（e-Tax推奨）

2026年は**e-Tax（オンライン申告）**が圧倒的に便利。税務署に行かなくていいし、還付も早いよ。

### 必要なもの

- マイナンバーカード（ICカードリーダー or スマホ）
- 源泉徴収票（本業がある人）
- 1年分の収入明細（サイトの支払調書）
- 経費の領収書・レシート
- 国保・年金の支払い証明書
- 銀行口座（還付金の振込先）

### e-Tax手順

**ステップ1**: 国税庁「確定申告書等作成コーナー」にアクセス

**ステップ2**: マイナンバーカードでログイン（スマホ認証もOK）

**ステップ3**: 収入を入力
- 給与所得（本業がある場合）
- 雑所得 or 事業所得（チャトレ・夜職）

**ステップ4**: 経費を入力
- 通信費: ◯◯円
- 消耗品費: ◯◯円
- 旅費交通費: ◯◯円 など

**ステップ5**: 控除を入力
- 基礎控除（自動）
- 社会保険料控除
- 医療費控除 など

**ステップ6**: 税額を確認して送信

**ステップ7**: 完了！還付がある場合は3〜4週間後に振込

### 「雑所得」vs「事業所得」どっち？

| 区分 | 基準 | メリット |
|---|---|---|
| **雑所得** | 副業・小規模 | 手続きが簡単 |
| **事業所得** | 継続的・本業レベル | 青色申告で控除増・損失繰越可 |

**目安**: 年間所得300万円超 or 専業なら事業所得を検討しよう💡

## H2: 住民税の「普通徴収」選択手順（本業にバレない方法）

副業が本業にバレる最大の原因は**住民税**。

### バレる仕組み

会社員の住民税は、給与から天引きされる（特別徴収）。
副業で所得が増えると住民税も増えて、会社の経理担当が「あれ？給与の割に住民税高くない？」と気づくわけ。

### 解決策: 普通徴収を選ぶ

確定申告時に「住民税に関する事項」欄で**「自分で納付（普通徴収）」**にチェックするだけ。

### 具体的な場所

**e-Taxの場合**:
1. 入力の最後の方「住民税等入力」画面
2. 「給与・公的年金等以外の所得に係る住民税の徴収方法」
3. **「自分で納付」にチェック** ✅

**紙の申告書の場合**:
- 第二表の左下「住民税・事業税に関する事項」
- 「自分で納付」に○印

### これだけで本業の給与天引き額は変わらない

副業分の住民税は、自宅に納付書が届くから自分で払うだけ。職場にはバレないよ💓

### 注意点

- 自治体によっては普通徴収を認めないケースもある（稀）
- 心配なら申告前に市区町村の税務課に電話確認しよう
- 「普通徴収にできますか？」と聞けばOK📝

## H2: 青色申告と白色申告の違い

事業所得で申告する人は、青色 or 白色を選べる。

| 項目 | 白色申告 | 青色申告（65万円控除） |
|---|---|---|
| **難易度** | ★☆☆ 簡単 | ★★★ 複式簿記必要 |
| **控除額** | なし | 最大65万円 |
| **損失繰越** | 不可 | 3年間繰越可 |
| **家族への給与** | 不可 | 経費にできる |
| **事前届出** | 不要 | 開業届+青色申告承認申請書 |

### 青色申告がオススメな人

- 年間所得200万円超
- 継続的に夜職・チャトレで働く予定
- 会計ソフト（freee、マネーフォワード）を使える

### 青色申告の始め方

1. **開業届**を税務署に提出（開業後1ヶ月以内）
2. **青色申告承認申請書**を提出（その年の3月15日まで）
3. 会計ソフトで日々の記帳
4. 翌年2〜3月に確定申告

### ナビちゃんの一言💓

初年度は白色でOK。慣れてきて「これは本業にしよう」と思ったら青色に切り替えよう。

会計ソフトは月980円〜で使えるから、65万円控除の節税効果を考えれば全然ペイするよ✨

## H2: 税理士に頼むべきケース

「自分でやるの無理…」と思ったら税理士に頼むのもアリ。

### 税理士に頼むべき人

- 年間所得500万円超
- 複数サイト・複数店舗で働いている
- 経費の仕訳が複雑すぎる
- 税務調査が入った
- 本業にバレたくない（プロの配慮あり）

### 費用相場

| 依頼内容 | 相場 |
|---|---|
| 確定申告のみ（スポット） | 5〜10万円 |
| 顧問契約（月次） | 月2〜5万円 |
| 税務調査対応 | 15万円〜 |

### 夜職・チャトレに理解のある税理士の探し方

- 「水商売 税理士」「風俗 税理士」でGoogle検索
- 女性税理士事務所を選ぶ（話しやすい）
- 初回無料相談を必ず使う

### 節税額 > 税理士費用 になればOK

税理士費用10万円で節税20万円できれば、差額10万円がプラス。
年収300万円超から真剣に検討しよう📝

## H2: よくある質問

### Q1: 申告しないとバレる？

**A**: バレる可能性は高いよ⚠️

サイト側は税務署に「支払調書」を提出している。
つまり「◯◯さんに年間いくら払いました」が税務署に把握されてる。
申告しないと数年後に「無申告加算税＋延滞税」で倍返しになるケースあり。

### Q2: 無申告がバレたらどうなる？

**A**: 以下のペナルティが発生する。

- **無申告加算税**: 本税の15〜20%
- **延滞税**: 年8.7%
- **悪質な場合は重加算税**: 本税の35〜40%

例: 本来10万円の税金を無申告 → 追加で5〜8万円払う羽目に😱

### Q3: 親の扶養から外れたくない…

**A**: 年間所得48万円以下に抑えれば扶養内キープ可能。

「収入」じゃなくて「所得」（収入-経費）が48万円以下ならOK。
計画的に経費計上・働く時間調整しよう💡

### Q4: 領収書を失くしたらどうすれば？

**A**: クレカ明細・銀行通帳・Amazon履歴で代用可能。

税務調査では「支払い事実が確認できればOK」とされている。
とはいえ領収書を残すのが一番安全だから、スマホ撮影を習慣にしよう📝

### Q5: 国民健康保険と国民年金は絶対払うべき？

**A**: 絶対払おう✅

- 国保未払い → 医療費10割負担
- 年金未払い → 将来もらえない+障害年金ももらえない

**経費にはできないけど「社会保険料控除」で全額控除できる**から、実質節税になるよ💓
