こんにちは、ナビちゃんだよ💓

高収入のお仕事に興味があっても、「安全なの？」「騙されない？」って不安だよね。
そこで、**お仕事を始める前に確認すべき安全チェックリスト50項目**をまとめたよ！

このチェックリストを使えば、危険な求人を避けて安全にお仕事ができるはず。保存して使ってね✨

---

## 【求人広告】チェックリスト（10項目）

求人を見つけた段階でチェックすべき項目だよ。

- [ ] 1. 運営会社名・所在地が明記されている
- [ ] 2. 給与の内訳が具体的に書かれている（「高収入」だけはNG）
- [ ] 3. 仕事内容が具体的に説明されている
- [ ] 4. 「絶対稼げる」「月100万確実」などの誇大表現がない
- [ ] 5. 連絡先が携帯電話番号だけでない（固定電話あり）
- [ ] 6. 求人サイトの口コミ・評価を確認した
- [ ] 7. SNSで店舗名を検索してトラブル情報がない
- [ ] 8. 「即日勤務可」「面接なしOK」だけをアピールしていない
- [ ] 9. 入店祝い金の条件が明確に書かれている
- [ ] 10. 退店・辞職の条件が明記されている

---

## 【面接・体験入店前】チェックリスト（10項目）

面接に行く前の準備段階のチェックだよ。

- [ ] 11. 面接場所が公共の場所またはビルのテナント
- [ ] 12. 面接日時を第三者（友人・家族）に伝えている
- [ ] 13. 面接場所の住所をGoogleマップで確認した
- [ ] 14. 身分証のコピーだけ求められている（原本預かりはNG）
- [ ] 15. 面接に行く前に質問リストを用意した
- [ ] 16. 体験入店の日給保証額を確認した
- [ ] 17. 交通費の支給有無を確認した
- [ ] 18. 服装の指定を確認した
- [ ] 19. 所要時間を確認した
- [ ] 20. 「友達と一緒に来て」と言われていない

---

## 【面接時】チェックリスト（10項目）

面接の場で確認すべき項目。違和感を感じたら断る勇気も大事だよ。

- [ ] 21. スタッフの対応が丁寧で威圧的でない
- [ ] 22. 給与体系を具体的に説明してくれる
- [ ] 23. 仕事の範囲（NGラインなど）を明確にしてくれる
- [ ] 24. 身分証の原本を預かろうとしない
- [ ] 25. 契約書の内容を読む時間をくれる
- [ ] 26. 「今日決めないと」と即決を迫らない
- [ ] 27. 前払い金や保証金を要求しない
- [ ] 28. 写真撮影の用途と管理方法を説明してくれる
- [ ] 29. 辞めたいときの手続きを説明してくれる
- [ ] 30. 他のスタッフやキャストの雰囲気が良い

---

## 【働き始めてから】チェックリスト（10項目）

実際に働き始めてからのチェック項目だよ。

- [ ] 31. 給与が約束通りの金額で支払われている
- [ ] 32. 給与明細をもらえている
- [ ] 33. 罰金・ペナルティ制度がない（または事前説明どおり）
- [ ] 34. シフトの変更・休みの申請が自由にできる
- [ ] 35. 身体的・精神的な無理を強要されない
- [ ] 36. お客様とのトラブル対応をスタッフがしてくれる
- [ ] 37. 防犯カメラ・セキュリティ対策がある
- [ ] 38. 衛生管理が徹底されている
- [ ] 39. 辞めたいと言ったら引き止められない
- [ ] 40. 個人情報が適切に管理されている

---

## 【身バレ・プライバシー】チェックリスト（5項目）

プライバシー保護は最重要。しっかり確認しよう。

- [ ] 41. 源氏名（芸名）を使える
- [ ] 42. 写真の加工・ぼかしが許可されている
- [ ] 43. SNSへの無断掲載がない
- [ ] 44. 個人のSNSと仕事アカウントを分けている
- [ ] 45. 自宅住所を教えなくてよい

---

## 【お金・税金】チェックリスト（5項目）

お金に関するトラブルを防ぐための確認項目。

- [ ] 46. 報酬の支払い方法（現金/振込）を確認した
- [ ] 47. 交通費・衣装代などの自己負担を確認した
- [ ] 48. 確定申告が必要なことを理解している
- [ ] 49. 経費にできるものを把握している
- [ ] 50. 収入を記録する習慣をつけている

> 確定申告について詳しくは [確定申告完全ガイド](/guides/kakuteishinkoku-guide) をチェック

## 🚨 こんな求人は危険！5つの警告サイン

### 1. 「月収100万円確実」など非現実的な金額を保証
→ どんなお仕事でも「確実」はありえない。

### 2. 面接で身分証の原本を預かろうとする
→ コピーの提示はOKだけど、原本預かりは絶対NG。

### 3. 「今日中に決めて」と即決を迫る
→ まともなお店は検討する時間をくれる。

### 4. 入店前にお金を要求される
→ 登録料・保証金・レッスン代などを先に払わせるのは詐欺の可能性大。

### 5. 仕事内容が曖昧で具体的な説明がない
→ 何をするか明確にしてくれないお店は避けよう。
